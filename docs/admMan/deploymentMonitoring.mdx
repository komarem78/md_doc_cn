## Unhealthy components

If your deployment sync has failed, you can monitor the unhealthy component logs to see what went wrong.
To find the logs section, you will need to:
1. Click on the selected component.
2. Go to the _LOGS_ section.
3. Select your application name.
4. There you will find stack trace for your specific component.
